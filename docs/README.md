# docs/ — Single Source of Truth (SSoT)

このディレクトリはBlenderMCPプロジェクト全体の公式ドキュメント群です。
仕様・ワークフロー・リファレンスはすべてここに集約し、唯一の情報源とします。

## 構成

| ファイル | 内容 |
|----------|------|
| `blender_mcp_reference.md` | BlenderMCP ツールリファレンス（13ツール詳細） |
| `workflow.md` | 標準ワークフロー（6ステップ） |
| `naming_conventions.md` | 命名規則・単位・座標系 |
| `use_case_checklists.md` | 用途別チェックリスト（3Dプリント/ゲーム/建築） |
| `troubleshooting.md` | トラブルシューティング |
| `project_template.md` | 子プロジェクトの作成手順とテンプレート説明 |

## 運用ルール

- ドキュメントの修正は `docs/` 内のファイルに対して行う
- 仕様書 (`blendermcp_spec.md`) は原本として保持し、docs/ が運用版
- 矛盾が生じた場合は docs/ を正とする
