# 新規プロジェクト開始

## ユーザー向け

何を作りたいか、自然に伝えてください。以下の情報があるとスムーズです：

- **何を作る？** — 例: ロボットのフィギュア、建物の外観、ゲーム用の剣
- **用途は？** — 3Dプリント / ゲーム / 建築 / その他
- **サイズや制約は？** — 例: 高さ15cm、ポリゴン5000以下
- **スタイルやイメージは？** — 例: デフォルメ、リアル、ローポリ

### 例

```
3Dプリント用にデフォルメのロボットフィギュアを作りたい。
高さ15cmくらい、丸っこい可愛い感じで。
```

```
ゲーム用のファンタジーな剣。ポリゴン少なめで、発光するエフェクト用の部分がほしい。
```

```
自宅のリビングを再現したい。幅5m×奥行4m×高さ2.5m。家具も含めて。
```

---

## Claude側 初動チェック（自動実行）

ユーザーの初回メッセージを受けたら、以下を実行する：

### STEP 0-1: 要件整理・確認

ユーザーの入力から以下を抽出し、**確認を返す**：

```
[要件整理]
- プロジェクト名: {snake_case で提案}
- 用途: {3dprint / game / architecture / general}
- 概要: {1-2行で要約}
- サイズ/スケール: {数値 + 単位}
- スタイル: {キーワード}
- 制約: {肉厚/ポリゴン数/精度 等}
- 不明点: {情報不足があれば質問}
```

不明点がある場合は質問する。なければ「この内容で進めてよいですか？」と確認。

### STEP 0-2: プロジェクト作成（承認後）

```bash
python scripts/new_project.py {project_name} --use-case {use_case}
```

### STEP 0-3: Blender準備

1. `get_scene_info` でBlender接続・状態確認
2. シーン初期化（`scripts/init_scene.py` 実行）
3. `models/wip/{name}_v001.blend` として初期保存

### STEP 0-4: 制作開始

→ 標準ワークフロー STEP 1〜6 へ移行（`docs/workflow.md` 参照）
